# WhatsApp Outreach Templates

Templates for common business communication scenarios. Adapt to fit the specific context and relationship.

## General Principles

- WhatsApp messages should be SHORT — 2-4 lines maximum
- Use the contact's first name
- Reference something specific to show it's not a mass message
- End with a clear question or call to action
- Match the language of previous conversations
- Use a friendly but professional tone
- Avoid links in the first message (triggers spam filters)

## Hebrew Templates

### פולואפ אחרי פגישה
```
היי [שם],
נהניתי מהשיחה שלנו על [נושא].
רציתי לבדוק אם הספקת לחשוב על מה שדיברנו — אשמח לקבוע שיחת המשך כשנוח לך.
```

### פולואפ אחרי הצעת מחיר
```
היי [שם],
רק רציתי לוודא שקיבלת את ההצעה ששלחתי ב-[תאריך].
אם יש שאלות או משהו שצריך התאמה — אני כאן.
```
### צ׳ק-אין עם לקוח קיים
```
היי [שם],
מה שלומך? רציתי לבדוק איך הולך עם [הפרויקט/השירות].
יש משהו שאנחנו יכולים לעזור בו?
```

### תזכורת תשלום (עדינה)
```
היי [שם],
רציתי לבדוק לגבי החשבונית מ-[תאריך] — רק רוצה לוודא שהכל תקין מצידכם.
אם יש משהו שצריך — אני כאן.
```

### פנייה ראשונית (חמה)
```
היי [שם],
אני מיכל מ-MSApps. ראיתי את [משהו ספציפי — פוסט, פרויקט, חיבור משותף].
יש לנו ניסיון ב-[תחום רלוונטי] ואשמח לספר בקצרה — מתאים לך שיחה קצרה?
```

## English Templates

### Follow-up After Meeting
```
Hi [Name],
Great chatting with you about [topic] earlier.
Just wanted to check if you had any thoughts — happy to set up a follow-up call whenever works.
```

### Proposal Follow-up
```
Hi [Name],
Wanted to follow up on the proposal I sent on [date].
Let me know if you have any questions or if anything needs adjusting.
```
### Client Check-in
```
Hi [Name],
Just checking in — how's everything going with [project/service]?
Anything we can help with on our end?
```

### Warm Introduction
```
Hi [Name],
I'm Michal from MSApps. [Mutual connection] suggested we connect —
we work with companies on [relevant area] and I'd love to share a few ideas.
Would you be open to a quick call?
```

### Payment Reminder (Gentle)
```
Hi [Name],
Just following up on the invoice from [date] — wanted to make sure everything is in order on your end.
Let me know if there's anything I can help with.
```

## Anti-Patterns — What NOT to Write

- Long paragraphs — if it's more than 4 lines, shorten it
- Multiple CTAs — one question/action per message
- Generic openers like "I hope this message finds you well" — too formal for WhatsApp
- Sending links in the first cold message
- Using ALL CAPS or excessive emojis
- Sending the same template to multiple contacts without personalizing
