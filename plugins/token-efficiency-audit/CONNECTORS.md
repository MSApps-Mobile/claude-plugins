# Connectors

## Required

- **Notion** — Stores audit reports, global rules page, and externalized skill data. Connect via Settings → Connectors → Notion (Cowork) or add Notion MCP to `.mcp.json` (Claude Code).

## Optional

- **Scheduled Tasks MCP** — Available by default in Cowork. Needed to audit and modify scheduled tasks.
- **Desktop Commander** — Used to inspect Mac-side skill files if VM mount is read-only. Not required if skills are accessible from the session.
