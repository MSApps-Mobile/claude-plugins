# Google Drive Upload Plugin — Pro License

Upload files from Claude directly to your Google Drive. Just say "upload to Drive" or "save this to Drive" — Claude handles the rest.

## Pro License includes

- ✅ **Unlimited uploads** — no monthly caps
- ✅ All file types: PDF, Word, Excel, images, video, zip, and more
- ✅ Folder targeting — save to any folder by name or ID
- ✅ Hebrew filenames supported
- ✅ Lifetime license — pay once, use forever
- ✅ All future updates included

## How to Use

After any file is created, just say:
- "upload to Drive"
- "save this to my Google Drive"
- "send to Drive"
- "תעלה לדרייב"

Claude will upload the file and give you a direct link.

---

## Setup (one-time)

1. Deploy the Google Apps Script (see setup guide)
2. Save your config to `~/.cowork-gdrive-config.json`:

```json
{
  "url": "https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec",
  "apiKey": "your-api-key"
}
```

Full setup guide: https://msapps.mobi/plugins
