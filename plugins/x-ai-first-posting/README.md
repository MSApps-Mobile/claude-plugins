# x-ai-first-posting

Write and publish posts to the **AI First Engineering** community on X (Twitter) for MSApps.

## What it does

This plugin helps Michal Shatz (@ShatzMichal) create and publish posts to the [AI First Engineering](https://x.com/i/communities/1896930451329032277) community on X — an Israeli tech community (~1.7K members) focused on transitioning from Human-First to AI-First engineering.

## Features

- **Tone-calibrated writing** — posts match the community's curious, humble, sharing voice (not marketing or guru energy)
- **Post type rotation** — rotates between sharing builds, observations, questions, discoveries, and before/after stories
- **Community-aware** — browses the feed before posting to avoid repeating topics
- **Notion integration** — checks strategy page and logs posted content
- **Scheduled posting** — Sunday and Wednesday at 10:00 AM Israel time
- **Approval workflow** — always gets Michal's approval before posting

## Requirements

- Claude-in-Chrome MCP (for posting to X)
- Notion MCP (for strategy tracking)
- X account with access to AI First Engineering community
