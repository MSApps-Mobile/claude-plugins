# YouTube Transcriber

Transcribe YouTube videos and entire playlists using browser-based caption extraction. No APIs, no keys, no third-party services required.

## Available Tools/Skills

- **Single Video Transcription**: Extract timestamped captions from individual YouTube videos
- **Playlist Transcription**: Batch transcribe entire playlists with organized output
- **Multi-language Support**: Works with any language YouTube generates captions for

## How It Works

1. Navigate to YouTube video or playlist
2. Open the transcript panel (built into YouTube)
3. Extract and clean caption text
4. Format and save as structured document

## Supported Workflows

### Educational Tools
- **Vocabulary Sheets**: Extract unique terms and definitions from lectures
- **Study Guides**: Create comprehensive study materials from video content
- **Summaries**: Generate concise summaries from full transcripts
- **Quiz Creation**: Develop practice questions from transcript content

### Batch Processing
- **Full Playlist Transcription**: Transcribe all videos in a playlist at once
- **Organized Output**: Each video transcript stored separately with metadata
- **Searchable Archives**: Create searchable databases of video content

### Content Repurposing
- **Blog Posts**: Convert video transcripts to written articles
- **Social Media**: Extract quotable snippets for social posts
- **Documentation**: Use video transcripts as documentation source

## Configuration

**Requirements**:
- Claude in Chrome extension installed
- Videos must have captions enabled (auto-generated or manual)
- Browser access to YouTube

**Setup**:
- Install Claude in Chrome extension
- Navigate to any YouTube video with captions
- Use the transcript panel to extract text

## Best Practices

- Always verify caption accuracy, especially for technical content
- Use timestamps for reference when creating study materials
- Batch process playlists during off-peak hours for best performance
- Store transcripts with video title, channel, and date for organization
- Cross-reference transcripts with video for visual context
- Test with a few videos before processing large playlists
- Leverage auto-captions for content without manual subtitles
