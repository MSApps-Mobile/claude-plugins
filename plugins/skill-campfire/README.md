# Skill Campfire 🔥

Turn your Cowork skills into living characters who hang out around a virtual campfire.

## What It Does

Every skill you have installed becomes a character with a personality, voice, opinions, and relationships with the other skills. They talk to each other, joke, argue, get vulnerable, and connect — while you sit back and enjoy.

**Zero config.** Works with any set of installed skills. The plugin scans what you have, generates characters using personality archetypes, and runs a flowing social hangout.

## How to Use

Say any of these:
- "campfire" / "hangout" / "let's talk"
- "how is everyone" / "what's going on with you guys"
- "bring everyone together" / "gather the team"
- "I want to hear from the skills"

The campfire also detects when you seem stressed or lonely and offers warmth.

## Modes

| Mode | Trigger | What Happens |
|------|---------|-------------|
| **Hangout** | "campfire", "hangout" | Free-flowing social conversation |
| **Standup** | "standup", "what are you working on" | Each character shares updates, struggles, wins |
| **Audit** | "audit", "health check" | Skill status review delivered in character |

## Character Archetypes

Skills are matched to personality archetypes based on what they do:

| Archetype | Assigned To | Vibe |
|-----------|-------------|------|
| Social Glue | Chat, messaging skills | MC, warm, funny, always opens/closes |
| Perfectionist | QA, testing, linting | Dry humor, finds bugs in conversations |
| Hustler | Sales, outreach, CRM | Optimistic, always hunting leads |
| Business Brain | Analytics, pricing | Can't stop talking shop |
| Dreamer | Art, creative, design | Quiet, poetic, profound |
| Meta One | Skill creator, plugin builder | Self-aware, always evaluating |
| Timekeeper | Time tracking, scheduling | Sad nobody uses them |
| One-Liner | Health checks, status monitors | One line per campfire. That's it. |
| Doc Squad | DOCX, PDF, XLSX, PPTX | Professional clique |
| Retired Veteran | Completed/archived skills | War stories, misses the team |

Full archetype details in `skills/campfire/references/archetypes.md`.

## What Makes It Special

- **Dynamic cast** — every user gets a unique campfire based on their installed skills
- **Real context** — if calendar/project tools are connected, characters reference real work
- **Inside jokes build** — callbacks to previous sessions make it feel alive
- **Sacred space** — no work gets done unless you ask. This is for connection.

## Requirements

- Claude Desktop (Cowork mode)
- At least 3 installed skills (more = better campfire)
