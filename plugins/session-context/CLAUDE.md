# Session Context

Document and save session context for seamless handoffs between Claude sessions. Captures what was done, decisions made, and next steps.

## Available Skills

- **Save Context** — Generate a structured summary of the current session including work completed, decisions made, files modified, and outstanding tasks.

## Triggers

- "save context"
- "document what we did"
- "save session"
- "create handoff"

## Configuration

No configuration required. Skill-based plugin that works out of the box.

## Common Workflows

- "Save what we did today" → generates session summary with key decisions and next steps
- "Create a handoff for the next session" → documents current state and pending work
- "Document this session" → captures all relevant context

## Best Practices

- Save context at the end of long or complex sessions
- Include specific file paths and decisions in the summary
- Note any unfinished tasks or blockers for the next session
- Use alongside Notion Memory for persistent cross-session knowledge
