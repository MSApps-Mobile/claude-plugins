# Claude in Chrome Fixer

This plugin was the original Chrome MCP connection repair tool. All functionality has been moved to the improved fix-chrome-connection plugin with better diagnostics and scheduling capabilities.

## Migration

If you were using claude-in-chrome-fixer, please switch to:
- **Plugin**: fix-chrome-connection
- **Same commands**: "fix chrome", "chrome not connecting"
