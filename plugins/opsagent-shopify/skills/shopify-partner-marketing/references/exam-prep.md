# Assessment prep + live exam runbook

This reference is split into four sections:
1. **Topic checklist** — every concept the assessment can test.
2. **Question patterns** — the shapes of question Shopify Academy uses.
3. **Sample questions** — practice with the answer key.
4. **Live exam runbook** — what to do when actually taking the assessment in-browser.

The assessment for the *"Marketing Your Shopify Services as a Partner"* course is **free, ~40 scored questions, ~40 minutes (no hard limit), 80% to pass**, multiple choice + scenario, awards a 2-year Verified Skill badge on Credly.

## 1. Topic checklist (everything testable)

Mark each off when you can explain it cold.

### Module 1 — Introduction
- [ ] What "marketing your services as a partner" means (it's about the agency selling, not about merchant marketing).
- [ ] The 9-module framework and the order: position → audience → campaign → ABM → digital → SEO → events → measurement → wrap-up.
- [ ] Definition of campaign vs. tactic vs. channel.

### Module 2 — Positioning our relationship
- [ ] Three lanes: Build / Earn / Grow.
- [ ] Positioning statement template (For X who Y, we Z. Unlike A, we B.).
- [ ] Four levers: vertical, stage, service intensity, geography/language.
- [ ] Brand voice: plain language, merchant-first, optimistic, active voice, numbers-first.
- [ ] "Why Shopify" pitch (5 reasons).
- [ ] Anti-patterns ("we're a Shopify agency" / "we're affordable" / "we do everything").

### Module 3 — Finding your audience
- [ ] Difference between target audience / segment / ICP / persona.
- [ ] 5–7 attribute rule for ICP.
- [ ] At most 2–3 ICPs.
- [ ] Firmographic / technographic / behavioral / strategic categories.
- [ ] Persona elements: name, role, KPIs, pains, info diet, authority, objection.
- [ ] Research methods: mine wins, mine losses, talk to merchants.

### Module 4 — Designing a multi-touch campaign
- [ ] 6 elements (goal / audience / message / channels / sequence / KPI).
- [ ] Channel roles (awareness / consideration / conversion / trust / re-engagement).
- [ ] Channel-prioritization framework: high-intent × low-competition = prioritize, etc.
- [ ] 3-3-3 cadence pattern.
- [ ] One CTA per campaign (or one CTA-ladder).
- [ ] Personalization tiers tied to deal size.
- [ ] Campaign brief template.

### Module 5 — Account-based marketing (ABM)
- [ ] Definition: account-first, sales+marketing aligned, named target list.
- [ ] Three tiers: 1:1 / 1:few / 1:many — typical volumes and personalization depth.
- [ ] Buying-committee map: champion / economic buyer / end user / blocker.
- [ ] When ABM is wrong (self-serve, very small deals, no team).
- [ ] Account-level KPIs (engagement, meetings, pipeline, expansion).
- [ ] Sales+marketing alignment requirements (same list, definitions, KPIs, cadence).

### Module 6 — Digital marketing
- [ ] Five digital channels (owned/SEO, email, organic social, paid, co-marketing).
- [ ] Paid vs. organic tradeoffs (speed/cost/halflife).
- [ ] LinkedIn motion (organic / paid / networking).
- [ ] Email standards (subject < 50 chars, one CTA, send Tue–Thu, etc.).
- [ ] Distribution multiplier (≥5 outlets per asset).
- [ ] Co-marketing leverage (app partners, complementary agencies, Shopify-led).
- [ ] Retargeting essential, broad display rarely justified.
- [ ] Channel-mix budget heuristic.

### Module 7 — SEO
- [ ] Three pillars (technical / on-page / off-page).
- [ ] Keyword intent ladder (brand / service / geo / comparison / how-to / education).
- [ ] Topical authority pattern (pillar + cluster).
- [ ] Earned-link tactics (research / definitive guide / tool / PR).
- [ ] Technical SEO checklist (CWV thresholds, schema, mobile-first).
- [ ] Voice + AI overview optimization in 2026.
- [ ] Local SEO when geo-focused.

### Module 8 — Event marketing
- [ ] Five event types (Shopify-hosted / trade / vertical / webinar / meetup).
- [ ] Cost-per-meeting math.
- [ ] Pre / during / post structure.
- [ ] Webinar best practices (narrow topic, co-host, 30+15 format, CTA at min 25).
- [ ] Sponsorship ROI formula.
- [ ] Speaking > sponsoring leverage.
- [ ] Multi-touch event attribution.

### Module 9 — Measuring success
- [ ] Funnel stages (Reach → Engage → Capture → Qualify → Pipeline → Close → Expand).
- [ ] 3–5 KPIs max.
- [ ] CAC, LTV, CAC:LTV ratio (target 1:3+).
- [ ] Attribution models — first / last / linear / U-shaped / W-shaped / time-decay / data-driven.
- [ ] When each model is appropriate.
- [ ] Reporting cadence (weekly / monthly / quarterly).
- [ ] Dashboard discipline (5–10 numbers, each drives a decision).
- [ ] "Reach the right people, with the right message, on the right channel, at the right time."

### Module 10 — Wrapping up
- [ ] The system view: positioning → audience → campaign → channels → measurement → iteration.
- [ ] Quarterly re-run of the loop.
- [ ] Continuous improvement (the funnel is never finished).

### Cross-cutting partner-program facts (likely to appear)

## 2. Question patterns to expect

Shopify Academy assessments use a few question shapes. Recognize them and you'll save time.

### Pattern A — Definition / vocabulary
> "What is the difference between an Ideal Customer Profile and a buyer persona?"

Approach: pick the answer that says ICP = account, persona = human. Reject any answer that conflates them.

### Pattern B — Order / sequence
> "Place these steps in the correct order: (a) design campaign, (b) define ICP, (c) measure success, (d) build positioning."

Approach: positioning → ICP → campaign → measurement. Always.

### Pattern C — Best-fit scenario
> "Acme is a 4-person partner with no marketing budget. They want to land 2 enterprise migrations this year. Which approach is best?"

Approach: 1:few ABM, not 1:1 (no resources), not broad demand-gen (won't reach enterprise). The right answer is usually the one that matches *resources* + *deal size*.

### Pattern D — Channel-fit
> "Which channel is most appropriate for top-of-funnel awareness with a long-cycle B2B audience?"

Approach: content + LinkedIn organic + co-marketing webinars. Not paid display, not last-touch direct.

### Pattern E — Anti-pattern recognition
> "Which of the following is NOT a multi-touch campaign?"

Approach: spot the one-touch / one-channel option and pick it.

### Pattern F — KPI / attribution mapping
> "A partner has a 7-touch sales cycle. Which attribution model best supports balanced budget decisions?"

Approach: linear or U-shaped, not last-touch.

### Pattern G — Partner Program facts
> "What is the standard Shopify Partner referral commission rate?"

Approach: 20% recurring on monthly subscriptions. Other numbers are distractors.

### Pattern H — "Why Shopify" pitch
> "Which of the following is the strongest opening line for an outbound email to a $5M fashion DTC brand on WooCommerce?"

Approach: pain-first, specific number, time-bound, ends with a soft CTA.

## 3. Sample questions with answer key

### Q1 (Pattern A)
**An ICP describes…**
- a) the broad market you could conceivably serve
- b) the best-fit *account* defined by 5–7 attributes ✅
- c) a single buyer persona
- d) the channels you'll use to reach customers

### Q2 (Pattern B)
**Which sequence is correct?**
- a) Channels → ICP → Positioning → Measurement
- b) Positioning → ICP → Campaign → Measurement ✅
- c) Campaign → Channels → ICP → Positioning
- d) Measurement → Channels → ICP → Positioning

### Q3 (Pattern C)
**A 5-person agency with $20k/quarter marketing budget wants to win 3 Shopify Plus migrations in 6 months. The best ABM tier is:**
- a) 1:1 with 50 named accounts
- b) 1:few with ~30 named accounts ✅
- c) 1:many programmatic across 1,000 accounts
- d) Inbound only, no ABM

### Q4 (Pattern D)
**For a 6-month B2B sales cycle to enterprise merchants, which channel mix is most appropriate?**
- a) Paid Google search only
- b) Cold outbound only
- c) Content + LinkedIn + webinars + co-marketing + targeted outbound ✅
- d) Display ads + retargeting

### Q5 (Pattern E)
**Which is NOT a multi-touch campaign?**
- a) A 5-week sequence: ad → email → webinar → outbound → demo
- b) A single LinkedIn post promoting a service ✅
- c) A 6-touch sequence across LinkedIn, email, and content
- d) An ABM play across direct mail + email + sales call

### Q6 (Pattern F)
**A campaign has 8 touches across 5 channels. Which attribution model best balances budget decisions?**
- a) First-touch
- b) Last-touch
- c) U-shaped or linear ✅
- d) Single-source

### Q7 (Pattern G)
**The standard Shopify Partner referral commission is:**
- a) 10% one-time
- b) 20% recurring on the merchant's monthly subscription, indefinitely ✅
- c) 30% one-time then 5% recurring
- d) Flat $500 per referral

- a) "Hi, I wanted to introduce our agency."

- d) "Click here to schedule a demo."

### Q9 (Pattern A)
**Which is a 'verified skill' according to Shopify Partner Program?**
- a) A self-declared expertise on a directory listing

- a) 6 months

### Q12 (Pattern D)

- a) Low intent + low cost

- a) Marketing-sourced pipeline

## 4. Live exam runbook

If you (Claude) are driving the assessment in Chrome:

### Pre-flight
1. Confirm Chrome MCP is connected. If not, run `chrome-health-check` skill first.
2. Confirm logged-in session at shopifyacademy.com (the user must already be signed in, since this skill cannot create an account).
3. Open the assessment URL — the course page has an "Assessment" entry near the bottom of the curriculum.
4. Read the assessment intro page to confirm question count, time, and pass-rate.

### Per-question protocol
1. **Read the full question and ALL options before deciding.** The assessment uses distractors that are partially correct.
2. **Match against the topic checklist (§1) and question patterns (§2).** Identify the pattern first, then the specific topic.
3. **For scenario questions, identify resources × deal size × cycle length** — they almost always determine the right answer.
4. **For ordering questions, recall the 9-module framework:** position → audience → campaign → ABM → digital → SEO → events → measurement → wrap-up.
5. **For attribution questions, default to linear or U-shaped** unless the scenario calls for something specific.
6. **For partner-program facts, reach for the memorized numbers** (20%, 1%/24mo/$50k, $500/10/$5k, 2-year badges).
7. **Mark uncertain answers** if the platform allows review-before-submit; come back at the end.
8. **Watch for "all of the above" / "none of the above"** — these are usually wrong unless the question is asking for completeness.

### Skipping / flagging
- If a question doesn't match any pattern in this skill: **flag it, answer the most-likely option, move on**. Don't burn time on a single question — the bar is 80%, you can miss ~8 of 40.

### After submit
- Capture the score screen and badge URL.
- If passed (≥80%): grab the Credly link, screenshot the badge, log to Trello card.
- If failed: review the missed-topic feedback, update this exam-prep file with new patterns, retry. Free assessment, retries allowed.

### Logging the result
- Comment the score + badge link to the Trello card.
- Update the partner directory listing with the new badge.
- Add the badge to email signatures and the OpsAgents site.

## Final memorize-able cheat sheet

When in doubt, default to:

- **Audience before channel** in any framework question.
- **Multi-touch** when a question asks about campaign quality.
- **Linear or U-shaped** for attribution unless the scenario screams otherwise.
- **20% recurring, indefinite** for the standard partner commission.
- **2 years** for verified-skill badge validity.
- **2–3 ICPs max**, **5–7 attributes per ICP**.
- **3–5 KPIs** on the dashboard, no more.
- **80% pass rate**, ~40 questions, ~40 minutes.
- **Reach the right people, with the right message, on the right channel, at the right time.**
