# YouTube Content Guide

## What This Skill Helps With
Writing YouTube metadata — titles, descriptions, tags, and chapter timestamps.
Not video scripts (though the user can ask for those separately).

## Title Writing
- **Length:** 60-70 characters (truncates after ~100)
- **Front-load keywords:** Put the main topic first
- **Patterns that work:**
  - How to [specific outcome]: "How to Deploy a Next.js App in 5 Minutes"
  - [Number] + [Topic]: "7 VS Code Extensions That Changed How I Code"
  - [Result] + [Method]: "I Built a SaaS in a Weekend — Here's the Stack"
  - Question: "Is TypeScript Actually Worth It in 2025?"
- **Avoid:** ALL CAPS, clickbait that doesn't deliver, emoji in titles

## Description Writing
The first 2-3 lines appear in search results and above the fold. Make them count.

### Structure
```
[1-2 sentence summary with keywords — this is what shows in search]

[2-3 paragraph expansion of what the video covers]

⏱ Timestamps:
0:00 - Intro
1:23 - [Topic 1]
3:45 - [Topic 2]
...

🔗 Links & Resources:
- [Link 1]
- [Link 2]

📱 Connect:
- Website: [URL]
- LinkedIn: [URL]
- Twitter/X: [URL]

#tag1 #tag2 #tag3
```

### SEO Tips for Descriptions
- Include primary keyword in first sentence
- Use natural language, not keyword stuffing
- 200-300 words minimum for SEO value
- Link to related videos/playlists

## Tags
- 5-15 tags
- Mix of: exact title match, broad topic, specific subtopics, common misspellings
- Primary keyword as first tag
- Don't use irrelevant popular tags — YouTube can penalize this

## Chapter Timestamps
- Start at 0:00 (required for chapters to work)
- Minimum 3 chapters, minimum 10 seconds each
- Use descriptive labels, not "Part 1, Part 2"
- Timestamps help both viewers and YouTube's algorithm

## Thumbnail Text (Suggestions)
When the user asks, suggest thumbnail text:
- 3-5 words maximum
- High contrast, readable on mobile
- Should complement (not duplicate) the title
- Create curiosity or promise value
