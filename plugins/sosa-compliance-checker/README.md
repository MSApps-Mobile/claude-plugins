# SOSA™ Compliance Checker

Audit your Claude plugins and skills against the SOSA™ methodology — Supervised Orchestrated Secured Agents.

## What it does

Scans your entire plugin ecosystem and evaluates each component against the four SOSA pillars:

1. **Supervised** — Are high-impact actions gated on human approval?
2. **Orchestrated** — Does the skill follow Plan→Act→Verify with structured outputs?
3. **Secured** — Are credentials safe? Is external data scanned for injection?
4. **Agents** — Is the role specification formal with clear boundaries?

## Usage

Just say: "Run a SOSA audit", "Check SOSA compliance", or "Audit my plugins"

## No setup required

Read-only plugin — only reads skill files and configs to analyze them.
