FAQ

Is this really free?
Yes — 100% free, no limits, no paywalls, no license keys. All plugins are MIT-licensed open source.
Do I need technical skills to install?
Not at all. Just paste the install command in Claude and you're done. Some plugins (like Google Drive) need a one-time 5-minute setup.
Does Google Drive Upload require an API key?
No API key needed. You deploy a simple Google Apps Script to your own Google account (takes 5 minutes), and the plugin handles the rest.
Can I contribute or suggest new plugins?
Absolutely! Open an issue or PR on our GitHub repo. We'd love community contributions.
Who made these?
MSApps — a boutique software studio specializing in mobile development, AI integration, and developer tools. We build these plugins to give back to the Claude community.
